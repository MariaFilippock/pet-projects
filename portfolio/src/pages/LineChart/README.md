# Interactive Line Chart

Визуализация данных с метриками Conversion Rate и интерактивным управлением

---

### Live Demo
https://mariafilippock.github.io/pet-projects/#/linechart  


### Source:  
https://github.com/MariaFilippock/pet-projects/tree/master/portfolio/src/pages/LineChart

---

## Features

### Core
✅ Conversion Rate line chart with percentage values  
✅ Tooltip с данными за день/неделю  
✅ Возможность выбора отображаемых линий (минимум одна всегда выбрана)  
✅ Автоматическая подстройка осей X/Y под видимые данные  
✅ Адаптивная верстка (671–1300 px)  
✅ Переключение режима: Day / Week  

### Additional
✅ Выбор типа графика: **Line / Smooth / Area**  
✅ Переключение темы: **Light / Dark**  
✅ Экспорт графика в PNG  
✅ Zoom / Reset Zoom

---

## Tech Stack

- **React 18** 
- **TypeScript**
- **Recharts** — визуализация данных с помощью графика
- **Ant Design** — набор UI-компонентов  
- **SCSS (препроцессор)** + **CSS Modules** (инкапсуляция стилей на уровне компонентов)

### Installation

```bash
git clone https://github.com/MariaFilippock/pet-projects.git

npm install

npm start

```

Открыть в браузере:

```bash
http://localhost:3000/pet-projects#/linechart
```
